# <span>Popup</span> <span>Property</span>



**Applies To:** [SysTrayItem](../objects/systrayitem.md), [ToolButton](../objects/toolbutton.md)

**Description**


The Popup property specifies the name of, or ref to, a (popup) [Menu](../objects/menu.md) object that is associated with a [SysTrayItem](../objects/systrayitem.md) or [ToolButton](../objects/toolbutton.md).


Note that Popup is ignored unless [Style](style.md) is set to `'DropDown'`.
