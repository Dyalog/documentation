# <span>TitleHeight</span> <span>Property</span>

**Applies To:** [Grid](../objects/grid.md)

**Description**

This property is a single number that specifies the height of the column titles displayed in a [Grid](../objects/grid.md) object. It is expressed in the units specified by the [Coord](coord.md) property of the [Grid](../objects/grid.md). The value `⍬` selects the default height.
