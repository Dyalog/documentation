# <span>Underline</span> <span>Property</span>



**Applies To:** [Font](../objects/font.md)

**Description**


This property specifies whether or not the characters in the font associated with a [Font](../objects/font.md) object are underlined or not. It is either 0 (normal) or 1 (underlined). There is no default; the value of this property reflects the underline characteristic of the font allocated by Windows.
