# <span>HelpFile</span> <span>Property</span>

**Description**

The HelpFile property is a read-only property that reports the pathname of a Windows help file associated with a particular COM object.

For an [OCXClass](../objects/ocxclass.md) or[ OLEClient](../objects/oleclient.md) object, the HelpFile property is read-only.

**Application**

Objects: [ActiveXControl](../objects/activexcontrol.md), [OCXClass](../objects/ocxclass.md), [OLEClient](../objects/oleclient.md)
