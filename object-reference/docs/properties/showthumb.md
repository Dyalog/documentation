# <span>ShowThumb</span> <span>Property</span>

**Description**

The ShowThumb property specifies whether or not the thumb in a [TrackBar](../objects/trackbar.md) object is visible. It is Boolean with a default value of 1 and it may be toggled on and off using `⎕WS`.

**Application**

Objects: [TrackBar](../objects/trackbar.md)
