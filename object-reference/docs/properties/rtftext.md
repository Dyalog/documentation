# <span>RTFText</span> <span>Property</span>

**Description**

The RTFText property is used to set or retrieve the contents of a [Clipboard](../objects/clipboard.md) or a [RichEdit](../objects/richedit.md) object in rich text format (RTF). It is always a character vector.

**Application**

Objects: [Clipboard](../objects/clipboard.md), [RichEdit](../objects/richedit.md)
