# <span>MaxSelCount</span> <span>Property</span>

**Description**

The MaxSelCount property specifies the maximum number of contiguous days that the user may select in a [Calendar](../objects/calendar.md) object.

MaxSelCount is an integer whose default value is 7.

MaxSelCount is ignored unless the [Style](style.md) property of the [Calendar](../objects/calendar.md) object is set to `'Multi'`.

**Application**

Objects: [Calendar](../objects/calendar.md)
