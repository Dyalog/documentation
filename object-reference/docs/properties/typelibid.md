# <span>TypeLibID</span> <span>Property</span>

**Description**

The TypeLibID property is a read-only property that reports the value of the globally unique identifier ([GUID](../miscellaneous/globally-unique-identifier-guid.md)) of the Type Library associated with a COM object.

**Application**

Objects: [ActiveXControl](../objects/activexcontrol.md), [OLEServer](../objects/oleserver.md)
