# <span>SetCellSet</span> <span>Method 171</span>

**Description**

The SetCellSet method sets the value of the [CellSet](../properties/cellset.md) property of a [Grid](../objects/grid.md) for a particular cell.

The argument to SetCellSet is a 3-element array as follows:

|-----|------|-------|
|`[1]`|Row   |integer|
|`[2]`|Column|integer|
|`[3]`|Value |0 or 1 |

**Application**

Objects: [Grid](../objects/grid.md)
