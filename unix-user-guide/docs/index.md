# About

The _Dyalog for Unix UI Guide_ covers the installation procedure and configuration options for Dyalog on a Unix-based platform. It details the environment variables that can be used to customise individual instances, the requirements for using the PuTTY terminal emulator, and the upgrade and uninstall procedures.describes the different input windows available and the keystrokes that can be used within them, how commands can be entered and the (configurable) keycode-keystroke mappings.

Assumed knowledge: A basic knowledge of UNIX.
