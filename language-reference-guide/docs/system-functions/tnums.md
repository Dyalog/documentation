# <span>Thread Numbers</span> `R←⎕TNUMS`{{key}}

`⎕TNUMS` reports the numbers of all current threads.

`R` is a simple integer vector of the base thread and all its living descendants.

<h2 class="example">Example</h2>
```apl
      ⎕TNUMS
0 2 4 5 6 3 7 8 9
```
