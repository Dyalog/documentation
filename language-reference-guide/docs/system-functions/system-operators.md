<h1 class="heading"><span class="name">System Operators</span></h1>

The following system facilities are for convenience implemented as operators rather than as functions:

|Name  |Description                   |
|------|------------------------------|
|`⎕R`  |Replace                       |
|`⎕S`  |Search                        |
|`⎕OPT`|Variant (Classic Edition only)|
