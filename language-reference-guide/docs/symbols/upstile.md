<h1 class="heading"><span class="name">Upstile</span> <span class="command">⌈</span></h1>

Monadic Upstile means
[Ceiling](../primitive-functions/ceiling.md)
```apl
      ⌈ 3.4 ¯3.4 3 0
4 ¯3 3 0
```

Dyadic Upstile means
[Maximum](../primitive-functions/maximum.md)
```apl
      1.1 ¯2 ⌈ 8.1 ¯3.4
8.1 ¯2

      ⌈/ 3 1 4 1
4
```
