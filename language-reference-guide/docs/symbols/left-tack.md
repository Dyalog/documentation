<h1 class="heading"><span class="name">Left Tack</span> <span class="command">⊣</span></h1>

Monadic Left Tack means
[Same](../primitive-functions/same.md)
```apl
      ⊣  1 2 3
1 2 3
```

Dyadic Left Tack means
[Left](../primitive-functions/left.md)
```apl
      'L' ⊣ 'R'
L
      ⊣/ 1 2 3
1
```
