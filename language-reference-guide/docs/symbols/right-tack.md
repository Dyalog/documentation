# <span>Right Tack</span> `⊢`

Monadic Right Tack means
[Same](../primitive-functions/same.md)
```apl
      ⊢  1 2 3
1 2 3
```

Dyadic Right Tack means
[Right](../primitive-functions/right.md)
```apl
      'L' ⊢ 'R'
R
      ⊢/ 1 2 3
3
```
