# Value Tips for External Functions

Value Tips can be used to view the syntax of external functions. If you hover over the name of an external function, the Value Tip displays its Function Signature.

For example, **Figure 1‑1** shows the mouse hovered over the external function `dt.AddMonths`, which reveals that it requires a single integer as its argument.

![s_value_tips_single](../../../img/s-value-tips-single.png)

Function signature – single integer argument

If an external function provides more than one signature, then they are all shown in the Value Tip (see **Figure 1‑2**; the function `ToString` has four different overloads.
